Política de Privacidade

1. Dados recolhidos

O PokéGuess não recolhe dados pessoais. O histórico de partidas é guardado localmente no teu browser através do localStorage e nunca é enviado para qualquer servidor.

2. Cookies

Este site não utiliza cookies. Todos os dados são guardados localmente no teu dispositivo.

3. API externa

Os dados dos Pokémon são obtidos através da PokéAPI (pokeapi.co), uma API pública e gratuita. Nenhum dado pessoal é enviado para esta API.

4. Termos de Utilização

O PokéGuess é um projeto académico sem fins comerciais. Pokémon e todos os personagens relacionados são propriedade da Nintendo, Game Freak e The Pokémon Company.
